# ✨ 电路寿命预测

### 在线演示
- 演示地址：
https://circuit-life-prediction.streamlit.app/

本项目是一个面向电子元器件与电路系统的寿命预测在线演示平台。 
核心能力：在热/电应力条件下，融合物理退化模型 与机器学习预测，实现从器件级退化、可靠度到电路级可靠性与电路级剩余使用寿命的流程。

> 适用场景：可靠性工程演示、课程教学、实验数据分析、工程团队评审与工具链参考实现。

## 功能概览

### ✅ 1）器件级：物理退化建模 + 参数拟合
- Excel 多工作表导入 、 侧边栏选择 sheet
- 数据在线编辑
- 预处理：
  - IQR（四分位距）异常值剔除
  - EWM（指数加权平均）平滑去噪
  - 支持 正常数据与增量型数据
- 物理退化模型参数拟合（SSA 搜索）
  - 拟合参数：`A, m, (E_a), p`
  - 可选：固定激活能 `E_a`

### ✅ 2）器件级：可靠度函数 R(t) 与失效概率 F(t)
- Monte Carlo：器件初值样本 `y0 ~ N(μ, σ)`
- 对每个初值样本求解寿命
- 对寿命样本拟合 Lognormal 分布，得到：
  - 可靠度函数：`R(t)`
  - 累计失效概率：`F(t) = 1 - R(t)`
  - `t_0.95`：当 `R(t) <= 0.95` 的首次时刻（等价于 `F(t) >= 0.05` 的首次时刻）

### ✅ 3）电路级：串、并、混合拓扑可靠性求解与结构图
- 串联、并联、混合
- 自动绘制简化结构图
- 输出：
  - 电路可靠度：`R_sys(t)`
  - 累计失效概率：`F_sys(t)`

常用公式：
- 串联：`R_sys(t) = Π_i R_i(t)`
- 并联：`R_sys(t) = 1 - Π_i (1 - R_i(t))`
- `F_sys(t) = 1 - R_sys(t)`

### ✅ 4）电路级：机器学习 RUL（物理-数据融合）
- 模型：**RNN 、 LSTM 、 Transformer**
- 物理-数据融合模式（可选）：
  1. 纯数据驱动
  2. 物理作为额外特征
  3. 残差学习：`y = y_phys + Δy_ml`
  4. 物理正则：`Loss = Loss_main + λ * MSE(y_pred, y_phys)`
- 超参数优化（可选）：
  - 贝叶斯优化 / 遗传算法（GA）/ 粒子群（PSO）
- 预测 → 阈值 → RUL：
  - 将预测退化曲线向未来滚动外推
  - 根据阈值触发方向（`y >= o` 或 `y <= o`）求得 失效时间
  - `RUL = t_fail - t_current`
- 电路级 RUL 聚合：
  - 串联：取 `min(RUL_i)`（最弱环节）
  - 并联：取 `max(RUL_i)`（冗余容错）
  - 混合：Stage 内并联取最大值，Stage 串联取最小值
